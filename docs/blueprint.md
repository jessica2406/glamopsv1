# **App Name**: GlamOps

## Core Features:

- Public Booking Flow: Multi-step booking process for customers: service selection, date/time slot booking (15-minute intervals), staff member assignment, and contact information collection.
- Salon Onboarding: Initial setup flow for salon owners: salon name input, logo upload, default buffer time configuration (10 minutes), and currency selection (â‚¹).
- Dashboard: Overview of daily operations: appointment summaries (25 appointments mock data), revenue tracking, calendar view, and appointment list.
- Calendar Management: Interactive calendar with drag-and-drop functionality for rescheduling appointments.
- Service Management: CRUD operations for salon services (6 services mock data): create, read, update, and delete services.
- Staff Management: Staff directory with profile photos, working hours, and role-based access control (owner vs. staff) (4 staff members mock data).
- Customer Management: Customer database with search functionality and appointment history tracking.
- Settings and Embedding: Settings page with embed code generation for website integration.

## Style Guidelines:

- Primary color: Soft purple (#8b5cf6) to align with user's theme request. Dark mode: fully supported.
- Background color: Light purple (#EDE9FE) with low saturation.
- Accent color: Slightly bluer purple (#6366F1) analogous to primary with higher brightness for contrast.
- Body and headline font: 'Inter' sans-serif for a clean and modern look, providing good readability and a neutral aesthetic.
- Use 'lucide-react' icons for a consistent and modern aesthetic across the application.
- Mobile-first design with responsive layouts using Tailwind CSS to ensure usability on all devices.
- Subtle transitions and animations using Tailwind CSS for a smooth and engaging user experience.
